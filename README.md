# leadersofdigital_interfax


## Требования:
Python 3.7
## Запуск проекта:
 Создайте виртуальное окружение:
``` virtualenv venv```  
Активируйте:
``` source venv/bin/activate ``` или ``` venv/Scripts/activate ```  
Установите зависимости: ``` pip install -r requirements```  
Запустите проект ``` python app.py ```
